# Good Knowledge Base Sample: dcp-front

This directory is a reference-quality sample for documenting a DCP front-end repository.

Use it as:

- a style and depth reference for `docs/knowledge/` output,
- a sample of how to attach source evidence to claims,
- a reminder to document route, screen, Vuex/DataStore, API, and change-playbook relationships together.

Do not use it as truth for another target project. When documenting a new project, Qwen must re-read the target repository and replace every claim and line reference with evidence from that repository.
