# Superpowers Unsupported Decision Log

This log records Superpowers upstream behavior that KIWI does not install into the Qwen3.5/qwencode closed-network runtime.

| Decision | Assets | Status | Reason | Replacement |
| --- | --- | --- | --- | --- |
| Public marketplace bootstrap | `.claude-plugin/*`, `.cursor-plugin/*`, `.opencode/*`, `gemini-extension.json` | REMOVE | KIWI cannot rely on public marketplaces or host-specific plugin commands. | Local Qwen extension generated by `backend/app/qwencode_runtime.py` and `scripts/build-offline-bundle.py`. |
| Visual brainstorming companion | `skills/brainstorming/scripts/*`, `skills/brainstorming/visual-companion.md`, `tests/brainstorm-server/*` | DEFER | Current Qwen3.5 image input is enabled through provider modalities. If the serving adapter rejects image media, KIWI has no closed-network visual companion service contract. | Text-first brainstorming, DOM/CSS/text evidence, screenshot paths, and human review checkpoints. |
| Other-platform test harnesses | `tests/claude-code/*`, `tests/opencode/*`, `tests/codex-plugin-sync/*` | REMOVE | These validate host platforms and plugin sync paths outside qwencode. | Deterministic trigger fixtures and local verifier scripts. |
| Release and repository governance files | `.github/*`, `scripts/bump-version.sh`, `scripts/sync-to-codex-plugin.sh`, icons, release notes | REMOVE | They do not affect Qwen runtime skill behavior and should not enter the offline bundle runtime extension. | Inventory accounting only. |
| Remote dependency installation | Any public installer, package install, or update flow implied by upstream docs/tests | REMOVE | Closed-network runtime must be reproducible from local bundle contents. | All Qwen skill folders are copied from `docs/superpowers-skills`. |

## Required Invariants

- DEFER and REMOVE decisions must never create a runtime instruction to fetch from the internet.
- FAST/lightwork must not receive superpowers skill content.
- Central knowledge docs remain the first context source for all retained skills.
- Selected `task_size` remains the source of truth in superpowers mode.
- `xsmall` remains Kiwi direct work; non-xsmall implementation mutation uses the selected implementation agent.
