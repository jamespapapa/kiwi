# Superpowers Full Port Inventory

## Scope

This inventory accounts for every file in `/Users/jules/Desktop/work/cpd-proxy/ref/superpowers-main` outside `.git`. The port target is KIWI superpowers mode on Qwen3.5-397B/qwencode in a Samsung Life closed network.

Runtime assets must be local. No external fetch, public marketplace bootstrap, remote update, or runtime package install is allowed. Central knowledge docs under `D:/aiops/docs/<project-key>/knowledge` are read first; optional Project Info Layer summaries under `D:/aiops/docs/<project-key>/project-info` are read only when present. Qwen3.5 image input is enabled through provider modalities. If the serving adapter rejects image media, visual workflows are represented as DOM/CSS/text evidence, screenshot paths, and human confirmation checkpoints.

## Decision Codes

| Code | Meaning |
| --- | --- |
| PORT | Rewritten as an individual Qwen skill or local KIWI runtime asset. |
| ADAPT | Behavior is preserved in condensed Qwen/KIWI form, usually inside a skill, deterministic fixture, or docs. |
| MERGE | Behavior is folded into existing KIWI runtime or offline bundle patching. |
| DEFER | Useful upstream behavior is intentionally held back because current Qwen3.5/qwencode closed-network constraints cannot support it safely. |
| REMOVE | Upstream file is platform, marketplace, governance, release, or public bootstrap material that must not be installed in KIWI runtime. |

## Closed-Network Decisions

- No external fetch at runtime. The offline bundle copies `docs/superpowers-skills/*/SKILL.md` into the Qwen extension from local source.
- Public plugin manifests are not installed. KIWI writes a local `qwen-extension.json` and local policy files.
- Hook behavior is merged into KIWI work-mode activation and pre-tool policy patches.
- Browser visual companion assets are DEFER until KIWI owns a closed-network local companion and DOM/CSS/text verification path.
- Model-calling upstream tests are replaced by deterministic trigger fixtures in `trigger-fixtures.json` and `scripts/assert-superpowers-full-port.py`.
- FAST/lightwork has no superpowers skill content. FAST remains direct and narrow.
